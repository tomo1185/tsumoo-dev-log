# フェーズ計画

## フェーズ0: 構想整理

サービス名、コンセプト、MVP 範囲、公開方針を整理します。

## フェーズ1: MVP

シンプルタスクと軽量な進捗付きタスクをすぐ作成し、気持ちよく完了し、直近の積み上げを振り返れる状態を作ります。

フェーズ1では、公開トップページ、認証、タスク管理、タスク一覧、達成状況・振り返り、アカウント設定を対象にします。

## フェーズ2: タスク管理拡張

カテゴリ、優先度、期限管理の高度化、PC 向け整理画面などを扱える状態を目指します。

## フェーズ3: 振り返り強化

週次/月次の振り返り、画像出力、PDF出力などを検討します。

## フェーズ4: 製品化検討

実際に使い続けられる完成度になった場合、製品化に必要な要件を整理します。
