# プロジェクト概要

## サービス名

Tsumoo（つもー）

## コンセプト

今日の「できた」を積もう。

## 概要

Tsumoo は、毎日の小さな ToDo や、仕事・学習で使う少し重いタスクを手軽に記録し、完了した行動を気持ちよく積み上げ、あとから振り返れるセルフマネジメントアプリです。

## 大事にする体験

- 入力は速い
- 完了は気持ちいい
- 振り返りは強い

## フェーズ1の方向性

フェーズ1では、機能を広げすぎず、チェック型 ToDo、軽量なステータス型 ToDo、ToDo 一覧、完了、直近7日間の振り返りに集中します。

ToDo 一覧では、すべて / 未完了 / 完了 のフィルターと、実施予定（開始） / 実施予定（終了） / 期限 の日付条件を扱います。

SNS共有、PDF出力、チーム機能などは将来機能として扱います。

## 個人情報の扱い

フェーズ1では、本名、住所、電話番号、生年月日、勤務先情報、公開プロフィール、ユーザー同士のやり取りは扱いません。

ログインに必要な最小限の情報と、ユーザー本人が登録する ToDo 情報のみを扱います。
